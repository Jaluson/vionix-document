# Vionix UI Concepts

## 1. 设计目标

为 Vionix 当前阶段优先输出两张核心界面：

1. `实时监控主屏`
2. `低代码仪表盘编辑器`

设计基调：

- 面向工业 IoT 场景，强调实时、稳定、专业
- 桌面 Web 优先，标准画板宽度 `1440px`
- 导航与数据区域强分层，避免信息拥挤
- 关键告警、连接状态、设备筛选始终可见

## 2. 视觉方向

### 2.1 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#F4F7FB` | 控制台浅灰蓝背景 |
| 深色导航 | `#0E1A2B` | 左侧主导航 |
| 主强调色 | `#1B66FF` | 当前菜单、主按钮、重点趋势 |
| 成功色 | `#16A34A` | 在线/正常状态 |
| 告警色 | `#E11D48` | 严重告警 |
| 警告色 | `#F59E0B` | 中等级异常 |
| 卡片背景 | `#FFFFFF` | 数据卡片、面板 |
| 边框色 | `#D7DEE7` | 面板边界、输入框 |
| 主文字 | `#0F172A` | 标题、重点数据 |
| 次文字 | `#64748B` | 描述、标签、辅助信息 |

### 2.2 字体

- 标题建议：`Inter` / `DIN` 风格无衬线
- 数据数字建议：较粗字重，突出读数变化
- 页面标题：`28-32px`
- 卡片标题：`20-24px`
- 常规正文：`14-16px`
- 辅助信息：`12-13px`

### 2.3 圆角与阴影

- 页面大卡片圆角：`24px`
- 次级容器圆角：`16-20px`
- 输入和按钮圆角：`16-20px`
- 阴影控制保守，更多用浅边框区分层级

## 3. 页面一：实时监控主屏

### 3.1 页面结构

```text
┌─ Sidebar 248 ─┬──────────────────────── Content 1152 ───────────────────────┐
│ Brand         │ Top Bar                                                     │
│ 实时监控      │ 设备监控中心 / 告警角标 / 设备筛选                          │
│ 仪表盘        ├──────────────────────────────────────────────────────────────┤
│ 设备目录      │ Hero Chart                                                  │
│ 告警中心      │ 秒级温度实时曲线 + 4 个实时状态胶囊                         │
│ 规则管理      ├───────────────┬──────────────────┬──────────────────────────┤
│ 设备分组      │ 分钟趋势       │ 小时趋势          │ 天趋势                    │
│ RBAC          ├──────────────────────────────────────────────────────────────┤
│               │ 告警列表 + 链路健康度                                       │
└───────────────┴──────────────────────────────────────────────────────────────┘
```

### 3.2 关键区块

#### A. 左侧导航

- 宽度 `248px`
- 深色背景
- 菜单项带小圆点或线性图标
- 当前页 `实时监控` 使用高亮胶囊底

推荐菜单顺序：

1. 实时监控
2. 仪表盘
3. 设备目录
4. 告警中心
5. 规则管理
6. 设备分组
7. RBAC

#### B. 顶部栏

- 高度约 `88px`
- 页面标题：`设备监控中心`
- 副标题：`实时监控 / 历史趋势 / 告警联动`
- 右侧放 `告警 3` 状态胶囊
- 设备筛选采用圆角选择框，默认示例 `sensor-001`

#### C. 实时曲线 Hero

- 深色大卡片，高度约 `388px`
- 主图表展示 `温度实时曲线`
- 标注数据来源：`秒级数据流 / WebSocket / 最近120点`
- 下方展示 4 个实时摘要：
  - `实时 25.3°C`
  - `最高 28.1°C`
  - `最低 22.5°C`
  - `连接 STOMP 正常`

#### D. 三列历史趋势

- `分钟趋势`：更偏表格/明细
- `小时趋势`：折线趋势
- `天趋势`：柱状统计

建议宽度比例：

- 分钟趋势：`300px`
- 小时趋势：`368px`
- 天趋势：`452px`

#### E. 告警与链路健康

- 左侧：最近告警列表
- 右侧：链路状态摘要
  - MQTT
  - WebSocket
  - InfluxDB
  - 规则引擎

## 4. 页面二：低代码仪表盘编辑器

### 4.1 页面结构

```text
┌──────────────────────────── 顶栏 1392 ────────────────────────────┐
│ 仪表盘编辑器 | 变量: 设备 sensor-001  时间 -2h | 预览 | 保存      │
├──── 左栏 264 ────┬──────────── 画布 780 ────────────┬── 右栏 330 ─┤
│ 组件库           │ 12列网格画布                      │ 组件配置     │
│ 折线图           │ 温度实时曲线                      │ 标题         │
│ 柱状图           │ 当前温度                          │ 数据源       │
│ 面积图           │ 告警列表                          │ 指标         │
│ 数字指标         │ 设备数据表格                      │ 时间粒度     │
│ 状态卡片         │ 能耗对比柱状图                    │ 颜色         │
│ 告警列表         │                                    │ 删除组件     │
│ 数据表格         │                                    │              │
└──────────────────┴───────────────────────────────────┴──────────────┘
```

### 4.2 关键区块

#### A. 顶栏

- 深色背景，区别于普通后台页
- 左侧显示 `仪表盘编辑器`
- 中间显示全局变量状态
- 右侧两个操作按钮：
  - `预览`
  - `保存`

#### B. 左侧组件库

- 白色卡片面板
- 纵向组件列表
- 图表类用浅蓝底区分
- 推荐首批组件：
  - 折线图
  - 柱状图
  - 面积图
  - 数字指标
  - 状态卡片
  - 告警列表
  - 数据表格

#### C. 中央网格画布

- 12 列浅灰网格
- 初始示例布局：
  - 大尺寸 `温度实时曲线`
  - 右上 `当前温度 25.3°C`
  - 左下 `告警列表`
  - 右下 `设备数据表格`
  - 底部全宽 `能耗对比柱状图`

#### D. 右侧配置面板

- 表单化配置项
- 默认字段：
  - 标题
  - 数据源
  - 指标
  - 时间粒度
  - 颜色
- 底部放 `删除组件` 危险操作按钮

## 5. 交互要点

### 5.1 实时监控

- 切换设备后清空秒级缓存并重订阅 WS
- 图表断线时出现显式状态提示，不只静默失败
- 告警状态颜色必须强提示，但避免整屏泛红

### 5.2 仪表盘编辑器

- 组件被选中后，画布中显示蓝色高亮边界
- 拖拽态建议显示栅格吸附线
- 变量修改后，预览组件统一刷新
- 保存按钮仅在有变更时高亮

## 6. 响应式建议

- 当前方案优先桌面端 `>= 1280px`
- 小屏笔记本场景下：
  - 左侧导航可收窄到 `88px`
  - 仪表盘编辑器左右栏支持折叠
  - 实时监控的三列趋势区改为两行布局
